    Get Started

    Command Line Interface

    On this page

    The JMS Bridge cli tool, jms-bridge, is available to perform some simple actions. Currently it supports sending and receiving of JMS text messages across a topic.

    To see all of the options use the included help system by executing jms-bridge help.
